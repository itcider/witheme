witheme는 한국어로 설정된 워드프레스에 최적화된 테마입니다.
제네레이트프레스를 기반으로 하였습니다.
gpl 라이선스를 따릅니다.

== 설명 ==

Wi theme는 워드프레스에 최적화된 테마입니다.
공식 문서는 오른쪽의 링크를 누르시면 됩니다.(https://wi.itcider.com).

== FAQ ==

= 무료인가요? =
네. 무료 테마입니다.

= 유료 버전이 있나요? =
아니요. 무료 버전만 있습니다.

= 공식 문서는 어디서 볼 수 있나요? =
오른쪽의 공식 문서 페이지에서 볼 수 있습니다.(https://wi.itcider.com/docs).

= 테마 옵션은 어디에서 설정 가능한가요? =
테마 옵션은 "외모 > 사용자 정의하기"에서 설정할 수 있습니다.
